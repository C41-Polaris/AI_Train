第一章·从零理解神经网络

深度学习的一切从这里开始。本章帮你建立关于神经网络最核心的直觉。

1.1 神经元与层

💡 一句话直觉

神经网络 = 一堆"神经元"（简单的数学函数）堆叠成"层"，层叠起来形成"深度"。每一层都在对输入做一次 非线性变换，让信息逐步抽象化。

一个神经元

一个神经元做的事其实很简单：

output = activation( w₁·x₁ + w₂·x₂ + ... + wₙ·xₙ + b )

其中：

  • x₁...xₙ — 输入（上一层传来的信号）
  • w₁...wₙ — 权重（这个神经元"记住"的参数）
  • b — 偏置（偏移量）
  • activation(...) — 激活函数（引入非线性）
x₁ ──╮ ├── w₁·x₁ x₂ ──┤ ╭───╮ ├── w₂·x₂ ─→ │ Σ │ ─→ activation(w·x + b) ─→ output x₃ ──┤ ╰───╯ ├── w₃·x₃ ... ──╯

全连接层（Feed-Forward Layer）

把多个神经元并排放置，每个神经元都连接到上一层的所有输出——这就是"全连接"（Fully Connected / Dense Layer）。

数学上，全连接层就是矩阵乘法 + 加偏置 + 激活函数：

output = activation( W · input + b )

其中 W 是权重矩阵（维度：output_dim × input_dim），b 是偏置向量。

Input layer Hidden layer Output layer [x₁] ─────╮ ┌───┐ ├────────────→ │ h₁│ ───────╮ ┌────┐ [x₂] ─────┤ └───┘ ├───────────→ │ y₁ │ ├──╮ ┌───┐ │ └────┘ [x₃] ─────┤ ├─────────→ │ h₂│ ───────┤ ├──╯ └───┘ │ ┌────┐ │ ┌───┐ ├───────────→ │ y₂ │ └─────────────→ │ h₃│ ───────╯ └────┘ └───┘

深度 = 多层堆叠

"深度学习"中的"深度"指的就是隐藏层的数量。层数越多，模型的"容量"越大——能学到更复杂的模式，但训练难度和过拟合风险也随之增加。

层数能力典型模型
0-1 层线性/简单非线性分类感知机、Logistic 回归
2-3 层任意函数逼近浅层 MLP
6-12 层特征抽象ResNet-18
20-80 层层次化特征ResNet-152 / GPT 系列
100+ 层超深表示LLaMA-3 (80层)

1.2 激活函数

如果没有激活函数，多层神经网络就退化成单层线性变换，再深也没区别。激活函数引入非线性，是神经网络"能学复杂模式"的根本原因。

Sigmoid

sigmoid(x) = 1 / (1 + exp(-x))

输出范围 (0, 1)，适合二分类输出层。

  • ✅ 输出可解释为概率
  • ❌ 饱和区梯度几乎为零（梯度消失）
  • ❌ 输出不是零中心的

ReLU（Rectified Linear Unit）

ReLU(x) = max(0, x)

现代深度学习默认选择。

  • ✅ 计算极快（只需取最大值）
  • ✅ 正半轴梯度恒为 1，缓解梯度消失
  • ✅ 天然稀疏激活（一部分神经元输出 0）
  • ❌ "神经元死亡"——大学习率下可能永久输出 0

GELU（Gaussian Error Linear Unit）

GELU(x) ≈ x · Φ(x) 其中 Φ(x) 是标准正态分布的 CDF

GPT、LLaMA 等现代 LLM 的首选。

  • ✅ 比 ReLU 更平滑，梯度更稳定
  • ✅ 自适应的门控行为（输入越小越倾向于关断）
  • ❌ 计算比 ReLU 稍贵

SwiGLU（Swish-Gated Linear Unit）

SwiGLU(x) = (x · sigmoid(x)) ⊙ (W₂ · x)

现代 LLM（如 LLaMA）的 FFN 层使用。

  • ✅ 门控机制让模型有更强的表达能力
  • ✅ 训练更稳定
  • ❌ 参数量比标准 FFN 多 50%
函数范围梯度使用场景
Sigmoid(0, 1)max=0.25, 易消失二分类输出层
Tanh(-1, 1)max=1, 易消失RNN / LSTM
ReLU[0, ∞)0 或 1CNN / 默认选择
GELU≈(-0.17, ∞)平滑Transformer / LLM
SwiGLU门控平滑+门控LLaMA / 现代 LLM

1.3 前向传播

前向传播 = 数据从输入层流到输出层的全过程。每一层的计算如下：

input x ↓ z¹ = W¹ · x + b¹ ← 第一层线性变换 a¹ = activation(z¹) ← 第一层激活 ↓ z² = W² · a¹ + b² ← 第二层线性变换 a² = activation(z²) ← 第二层激活 ↓ ...（更多层） ↓ ŷ = output_layer(aᴸ) ← 输出层预测 其中 L = 层数，ŷ = 模型预测值

前向传播本质就是一连串的矩阵乘法 + 激活函数。每条路径的权重 W 和偏置 b 就是模型需要学习的全部参数。

1.4 损失函数

损失函数衡量"模型预测 ŷ 和真实值 y 之间的差距"。目标就是让损失最小化。

均方误差（MSE）

MSE = (1/n) · Σ(yᵢ - ŷᵢ)²

用于回归任务（预测连续值，如温度、房价）。

  • ✅ 凸函数，容易优化
  • ❌ 对异常值敏感（平方放大误差）

交叉熵（Cross-Entropy）

CE = - Σ yᵢ · log(ŷᵢ) （分类任务）
CE = - [y · log(ŷ) + (1-y) · log(1-ŷ)] （二分类特殊情况）

用于分类任务（图像分类、文本分类等），也是 LLM 预训练用的损失函数。

  • ✅ 配合 Softmax 效果极好
  • ✅ 模型"不确定"时惩罚很大
  • ✅ LLM 训练中 = 预测下一个 token 的交叉熵

🔑 LLM 的损失函数

LLM 预训练中，损失函数是"下一个 token 预测"的交叉熵：

Loss = -(1/N) · Σ log P(token_t | token_<t)

模型在每一步预测下一个 token 的概率分布，然后与真实的下一个 token 算交叉熵。这个值的指数就是 perplexity（困惑度）。

1.5 反向传播

反向传播（Backpropagation）是神经网络训练的核心算法。它利用链式法则，从输出层向输入层逐层计算每个参数对损失的梯度（偏导数）。

🧮 链式法则

∂Loss / ∂w = (∂Loss / ∂ŷ) · (∂ŷ / ∂z) · (∂z / ∂w)

每一层的梯度都可以分解为"后面传回来的梯度" × "本层局部梯度"。

Loss ↑ ∂L/∂ŷ ← 损失对输出的梯度 ↑ ∂L/∂wᴸ = ∂L/∂zᴸ · ∂zᴸ/∂wᴸ ← 第 L 层权重梯度（计算并保存） ↑ ∂L/∂aᴸ⁻¹ = ∂L/∂zᴸ · ∂zᴸ/∂aᴸ⁻¹ ← 梯度向上一层传播 ↑ ∂L/∂wᴸ⁻¹ = ∂L/∂zᴸ⁻¹ · ∂zᴸ⁻¹/∂wᴸ⁻¹ ← 第 L-1 层 ↑ ...（逐层向前） ↑ ∂L/∂w¹ ← 第一层权重梯度

每一步需要保存前向传播时的中间结果（如激活值 a），所以训练时 显存占用 > 推理时。这也是 Gradient Checkpointing 等技术想要解决的问题。

一次完整训练迭代

1. 前向传播 ──→ 计算预测 ŷ 2. 计算损失 ──→ Loss = CE(ŷ, y) 3. 反向传播 ──→ 计算每层 ∂Loss/∂w 4. 梯度下降 ──→ w = w - lr · ∂Loss/∂w 5. ↺ 重复步骤 1

1.6 优化器

有了梯度，怎么更新参数就是优化器的事。不同的优化器本质上是不同的"梯度更新策略"。

SGD（随机梯度下降）

w_{t+1} = w_t - η · g_t 其中 g_t = ∂Loss/∂w_t
  • ✅ 理论上收敛性质好，泛化能力强
  • ❌ 收敛慢，对学习率敏感
  • ❌ 在鞍点/峡谷地形中震荡严重

SGD + Momentum（动量）

v_t = β · v_{t-1} + (1-β) · g_t
w_{t+1} = w_t - η · v_t

Adam（Adaptive Moment Estimation）

当前最广泛使用的优化器。它结合了 Momentum 和 RMSProp 两者的优点。

g_t = ∂Loss/∂w_t ← 当前梯度 m_t = β₁·m_{t-1} + (1-β₁)·g_t ← 一阶动量（梯度的均值） v_t = β₂·v_{t-1} + (1-β₂)·g_t² ← 二阶动量（梯度方差） m̂_t = m_t / (1-β₁^t) ← 偏差校正 v̂_t = v_t / (1-β₂^t) w_{t+1} = w_t - η · m̂_t / (√v̂_t + ε)
  • ✅ 自动调整每个参数的学习率
  • ✅ 对超参数不太敏感，默认值通常就够
  • ✅ 适合稀疏梯度（NLP 任务常见）
  • ❌ 需要更多显存（存储 m_t 和 v_t，参数量 ×2）

AdamW（Adam with Decoupled Weight Decay）

AdamW 是 Adam 的一个关键改进——把权重衰减（L2 正则化）从梯度更新中解耦出来。

Adam 方式: g_t = ∂Loss/∂w_t + λ·w_t ← 在梯度里加 L2 AdamW 方式: Update w_t using Adam, then add weight decay separately w_{t+1} = w_t_adam - η·λ·w_t ← 在优化器外部加 L2
  • ✅ 更稳定的训练（解耦了 learning rate 和 weight decay）
  • ✅ 现代 LLM 训练的标准选择
  • ⚠️ Hugging Face Transformers 的默认优化器已经是 AdamW

📌 优化器选择建议

LLM 微调：AdamW（默认 β₁=0.9, β₂=0.999, lr=2e-4~5e-5）
从头预训练：AdamW + cosine 学习率调度 + warmup
极简/大 batch：SGD + Momentum（泛化性能更好但调参困难）
如果显存紧张：Adafactor（省了优化器状态的显存）

1.7 总结

📌 核心要点

  1. 神经网络 = 矩阵乘法 + 激活函数的层层堆叠
  2. 非线性 来自激活函数，没有它就只是一层线性变换
  3. 反向传播 用链式法则高效计算梯度
  4. 优化器 决定了网络参数怎么更新，Adam/AdamW 是现代首选
